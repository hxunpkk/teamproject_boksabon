## **📝 포트폴리오 개요**
<img width="100%" alt="메인 페이지" src="https://user-images.githubusercontent.com/67828187/279621773-f4c14007-701d-4050-8ebb-05f9b3b0d321.JPG" />

> **프로젝트:** 여행지 관련 데이터 제공 사이트
>
> **분류:** 팀 프로젝트
>
> **제작 기간:** 약 3주
>
> **주요 기능:** 반응형 웹, 날씨 데이터 출력 (openweatherAPI), 의상 추천, 지역 별 음식점 추천(카카오맵 API), 여행지 추천, 미세먼지 그래프 
>
> **사용 기술:** HTML, CSS, JavaScript, JQuery
>
